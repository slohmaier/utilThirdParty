# utilThirdParty

Third-party library collection for wxWidgets-based App Store applications.

## Libraries

- **wxWidgets** - Cross-platform GUI framework with accessibility support
- **OpenCV** (planned) - Computer vision library

## Quick Start

```bash
pip install scons
scons
```

## Configuration

See `config.default.json` for available options. Projects using this repo provide a `utilThirdParty.json` to specify libraries and overrides.
